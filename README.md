## Setup

Create an `.env` file:

```sh
$ cp .env.example .env
```

Edit this file and set the values for the requested environment variables, example:

```ts
NEXT_PUBLIC_HOST_API = 'api url';
```

## Run app

```sh
$ pnpm install
```

```sh
$ pnpm run dev
```

## Build app

```sh
$ pnpm run build
```
